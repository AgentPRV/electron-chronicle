A simple Electron-based Todo application with features to add, delete, edit, and view notes.

## Features

- Add new notes with a title and description.
- Edit existing notes.
- Delete notes.
- View a list of all notes.

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

- Node.js: [Download and Install Node.js](https://nodejs.org/)
- npm: [Download and Install npm](https://www.npmjs.com/get-npm)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/todo-app.git
   ```
2. Navigate to the project directory:
    ```bash
    cd electron-chronicle
    ```

3. Manually make one file as `notes.db`
    ```bash
    type notes.db
    ```
4. Install dependencies:
    ```bash
    npm install
    ```
### Usage

1. Run the application:
    ```bash
    npm start
    ```
2. The Todo app window will open, allowing you to add, edit, delete, and view notes.

### After Installation
1. After launching the Todo app, you can start adding new notes by clicking the "+ Add Note" button.
2. To edit a note, click the pencil icon next to the note you want to edit.
3. To delete a note, click the trash icon next to the note you want to delete.
4. All your notes will be listed in the main window for easy reference.
